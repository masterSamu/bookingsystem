# bookingsystem
Booking system for online shop.

My client has an online shop and she received a lot of booking requests for products from customers via email or facebook. She needed a system to store those bookings.

# Techs used
- Java (Servlets)
- MySQL
- JPA
- JavaScript, (JQuery)
- HTML, CSS, (JSP)
- Google App Engine
- Google Cloud SQL
